样本随机采样

有时候拿到的数据集中有很多个样本，但如果用所有的数据一起训练，由于计算机性能的限制会花费很多时间。或者想要先在一部分数据集上实验哪一种算法得到的效果更好，可以使用pandas.DataFrame.sample 进行随机抽样，先在一部分数据集上做试验。

sample(n=None, frac=None, replace=False, weights=None, random_state=None, axis=None)
n=None 期望抽取的样本数量。当frac=None默认为1
frac=None 期望抽取的样本占所有样本的百分比
！n / frac 只要设置其中一个就好了
replace=False 抽样放回还是不放回
weights=None
random_state=None 随机数生成器种子。int or numpy.random.RandomState
axis=None 样本的轴 int or string, optional
